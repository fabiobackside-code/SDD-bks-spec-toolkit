# 🛠️ Guia de Estudo: BKS Spec Toolkit (`toolkit-app`)

Bem-vindo ao roteiro de estudos focado no **BKS Spec Toolkit**.
Este não é apenas um material sobre a teoria do SDD (Spec-Driven Development), mas sim um **passo a passo** prático para operar a ferramenta que materializa esses conceitos (`toolkit-app`).

## 📚 Índice de Aprendizado

Siga a trilha na ordem para o melhor aproveitamento:

1. **[01. Introdução ao Toolkit](./01-introducao.md)**
   Descubra o que é a ferramenta, qual o papel dela no ecossistema e porque automatizar o SDD é um divisor de águas.

2. **[02. Instalação e Setup](./02-instalacao-e-setup.md)**
   Como baixar as dependências, preparar o Node.js e realizar o build da CLI (Command Line Interface).

3. **[03. Criando seu Primeiro Projeto](./03-criando-projetos.md)**
   Um walkthrough prático utilizando o comando de criação para gerar uma fundação limpa (Agnóstica ou .NET).

4. **[04. Anatomia da Estrutura Gerada](./04-estrutura-gerada.md)**
   Entendendo cada pasta estruturada pelo Toolkit (`docs/`, `specs/`, arquivos base) e as funções dos arquivos `CLAUDE*.md`.

5. **[05. O Ciclo de Vida Prático no SDD](./05-ciclo-de-vida-sdd.md)**
   Como você e o agente de IA utilizam a estrutura criada no dia a dia para desenvolver features novas de ponta a ponta.

---

**Pré-requisitos básicos**:
- Node.js instalado na máquina.
- Compreensão básica do que é SDD (veja as referências na raiz de `/learning` do repositório).
- O código do toolkit disponível na pasta `toolkit-app`.
