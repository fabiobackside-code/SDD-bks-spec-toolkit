# product.md — Agent Generator

## O que é
CLI para criação de agentes de IA customizados sem necessidade de programação.
O usuário descreve em linguagem natural o que o agente deve fazer,
e o sistema gera um agente funcional pronto para uso.

## Problema que Resolve
Criar agentes de IA hoje requer: escrever system prompts, definir ferramentas,
implementar loops de execução, tratar erros — tudo manualmente.
O Agent Generator abstrai isso: o usuário descreve a intenção, o sistema gera o agente.

## Usuários
- **Desenvolvedores:** que querem prototipagem rápida de agentes
- **Equipes de produto:** que querem testar agentes antes de investir em desenvolvimento
- **Consultores de IA:** que precisam demonstrar capacidades de agentes para clientes

## Casos de Uso Principais
1. **Agente de Suporte:** responde perguntas com base em contexto do negócio
2. **Agente de Análise:** processa documentos e retorna insights estruturados
3. **Agente de Automação:** executa workflows com ferramentas (busca, cálculos, APIs)
4. **Agente de Pesquisa:** coleta e sumariza informações de múltiplas fontes

## Proposta de Valor
"Em 5 minutos, do zero a um agente de IA funcional — sem escrever código de agente."

## Métricas de Sucesso
- Tempo de criação de agente: < 5 minutos
- Agente gerado é funcional sem modificações na primeira tentativa: > 80% dos casos
- Custo médio por sessão de agente: < $0.10 (usando modelos eficientes)
